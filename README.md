#  E-Commerce Web Application

A full-stack E-Commerce web application built using Java and Spring Boot.  
This project demonstrates secure authentication, role-based access control, and complete product & order management.

---

##  Tech Stack

### Backend
- Java
- Spring Boot

### Security
- Spring Security

### Frontend
- Thymeleaf
- HTML
- CSS

### Database
- MySQL

### ORM
- Spring Data JPA (Hibernate)

### Build Tool
- Maven

---

##  Key Features

- Secure User Registration & Login
- Role-Based Authorization (Admin & User)
- Product Listing & Management
- Add to Cart & Order Placement
- Admin CRUD Operations

---

##  Project Structure

Controller → Service → Repository → Database

Follows a clean layered architecture for scalability and maintainability.

---

##  How to Run

1. Clone the repository:

git clone https://github.com/manjunath-bc/E-Commerce-web-Application.git

2. Configure database credentials in `application.yml`

3. Run the project as a Spring Boot Application

---

##  Objective

To build a secure, scalable, and production-ready E-Commerce application using modern Java Full Stack technologies.
